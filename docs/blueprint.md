# **App Name**: Zenith Portfolio

## Core Features:

- Hero Section: Display name, technical identity, and a call to action.
- Project Showcase: Highlight 3-5 key technical projects with descriptions and links.
- Skills Summary: Display technical skills and experience in a concise format.
- Extracurricular Highlights: Showcase creative depth through extracurricular activities like photography.
- About Section: Provide a brief, professional bio focusing on problem-solving approaches.
- Contact Links: Include links to email, GitHub, and LinkedIn.

## Style Guidelines:

- Primary color: Muted dark blue (#455A64) for calmness and intelligence.
- Background color: Near-black (#121212) for a sleek, professional feel.
- Accent color: Soft gray-blue (#607D8B) to provide subtle highlights.
- Body and headline font: 'Inter', a sans-serif font known for its clean, modern design, used for both headings and body text, emphasizing clarity and precision.
- Use heavy whitespace and a strict grid system to ensure a minimal, uncluttered layout.
- Implement subtle micro-interactions with animations under 250ms, focusing on hover effects like scaling (1.02) or opacity shifts, to add a touch of dynamism without distraction.
